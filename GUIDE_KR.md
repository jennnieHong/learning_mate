# 📔 LearningMate 사용자 가이드

LearningMate는 사용자 가 가진 데이터를 문제집으로 변환하여 효율적으로 암기하고 학습할 수 있도록 돕는 도구입니다.

---

## 1. 시작하기: 파일 업로드

학습을 시작하려면 먼저 문제 데이터가 담긴 파일을 업로드해야 합니다.

### 지원 파일 형식
- **Excel (.xlsx)**: 정석적인 표 구조 (권장 ⭐)
- **CSV (.csv)**: 쉼표로 구분된 데이터
- **TXT (.txt)**: 탭(Tab)으로 구분된 데이터

### 파일 구조 가이드
- **열 1**: 문제 또는 설명 (예: apple)
- **열 2**: 정답 (예: 사과)
- **열 3 이후**: 선택지 (객관식으로 풀 경우 나타나는 보기들)

> [!TIP]
> 첫 번째 행은 제목(헤더)으로 간주되어 데이터에서 제외됩니다.

---

## 2. 학습 모드 활용하기

문제집을 클릭하면 학습을 시작할 수 있습니다.

### 📚 설명 모드 (Flip Card)
- 카드의 앞면을 보고 정답을 머릿속으로 떠올려 보세요.
- 카드를 클릭하면 뒤집히면서 정답이 나타납니다.
- **[완료]**: 아는 문제는 완료 처리하여 다음 학습에서 제외할 수 있습니다.
- **[미완료]**: 아직 헷갈리는 문제는 다음에 다시 나타납니다.

### ✍️ 문제 모드 (Quiz)
- 객관식 선택지 중 정답을 고릅니다.
- 정답 여부에 따라 오답 횟수가 기록되며, 이는 나중에 '오답 노트'에서 확인할 수 있습니다.

---

## 3. 스마트 오답 노트 (복습하기)

LearningMate의 가장 강력한 기능은 **오답 모아보기**입니다.

1. 메인 화면에서 복습하고 싶은 파일들을 **체크박스**로 선택합니다 (전체 선택 가능).
2. 선택된 파일들에 틀린 문제가 있다면 상단에 **오답 노트 대시보드**가 나타납니다.
3. **'오답 학습 시작하기'** 버튼을 클릭하면 여러 파일에 흩어진 오답들만 모아서 한 번에 공부할 수 있습니다.

---

## 4. 문제 편집 및 관리

### ✏️ 직접 만들기/수정하기
- **[새로 만들기]**: 외부 파일 없이 웹에서 직접 문제를 입력하여 문제집을 만들 수 있습니다.
- **[편집]**: 업로드한 문제의 오타를 수정하거나, 진행 상황(완료 여부, 오답 횟수)을 개별적으로 초기화할 수 있습니다.

### 🗑️ 휴지통
- 실수로 삭제한 파일은 휴지통에서 언제든지 복원할 수 있습니다.
- 영구 삭제를 원할 경우 휴지통에서 비우기를 진행하세요.

---

## ⚙️ 사용자 설정

설정 화면에서 다음 옵션을 조정하여 나만의 학습 환경을 만드세요:
- **학습 순서**: 등록된 순서대로 풀기 vs 무작위로 풀리기(추천 ⭐)
- **질문 방향**: 설명 보고 정답 맞히기 vs 정답 보고 설명 맞히기
- **반복 모드**: 모든 문제를 맞힐 때까지 무한 반복
